# 介绍
2024年6月我接触到了tailscale,后在个人博客上发布了derper与headscale的搭建教程   
2024年9月8日headscale-Admin首个版本正式开源发布

# 功能
- 用户管理
- 用户自行注册
- 用户到期管理
- 流量统计
- 基于用户ACL
- 节点管理
- 路由管理
- 日志管理
- 预认证密钥管理
- 角色管理
- api和menu权限管理
